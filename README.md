# ❤️ Heart Attack Risk Prediction using Optimized Decision Tree

## 📌 Overview

This project predicts the likelihood of a heart attack based on patient medical and lifestyle data. A Decision Tree Classifier is used and further improved using hyperparameter tuning and cross-validation to enhance model performance and generalization.

## 🧠 Model Used

* Decision Tree Classifier (Scikit-learn)

## ⚙️ Workflow

1. Data loading and preprocessing
2. Handling missing values
3. Encoding categorical variables
4. Exploratory data analysis
5. Train-test split
6. Initial model training using Decision Tree
7. Hyperparameter tuning (e.g., depth, splitting criteria)
8. Cross-validation for model validation
9. Final model evaluation

## 📊 Features Used

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise-Induced Angina
* ST Depression (Oldpeak)

## 🔧 Model Optimization

* Hyperparameter tuning applied to improve model performance
* Cross-validation used to ensure model generalization and reduce overfitting

## 📊 Evaluation Metrics

* Accuracy Score
* Cross-validation score
* Confusion Matrix 

## 📈 Result

The optimized decision tree model demonstrates improved performance compared to a baseline model, showing the importance of tuning and validation in machine learning workflows.
